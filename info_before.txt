零界点（WindowAutomationControlTool）安装说明
=========================================

感谢您选择使用 “零界点（WindowAutomationControlTool）”。

在开始安装之前，请仔细阅读以下信息，以便更好地使用本软件。

一、软件简介
--------------------------------
零界点 是一个运行在 Windows 平台上的窗口自动化工具，主要特点包括：

1. 多项目管理：
   - 针对不同目标窗口创建独立项目（如：游戏、业务系统、工具软件）；
   - 支持多个项目同时运行。

2. 多场景支持：
   - 每个项目下可以定义多个场景（如：登录界面、主界面、结算界面）；
   - 支持为每个场景配置识别锚点，实现稳定的自动识别。

3. 自动化操作编排：
   - 支持鼠标点击、拖拽、键盘按键、文本输入、等待等操作；
   - 使用相对坐标，适配不同分辨率和窗口大小；
   - 可以调整操作顺序、启用/禁用单个操作。

4. 后台执行：
   - 使用 Windows 消息（PostMessage）在后台执行操作；
   - 不移动真实鼠标、不抢占键盘输入；
   - 可以在正常使用电脑的同时自动执行脚本。

二、运行环境
--------------------------------
1. 操作系统：
   - Windows 10 / Windows 11（推荐 64 位系统）

2. 依赖组件：
   - 本安装包已内置运行所需的 Python 运行环境和相关库；
   - 安装时无需额外安装 Python。

3. 注意事项：
   - 如需对某些以“管理员权限”运行的程序进行自动化操作，请以管理员身份运行本软件；
   - 建议关闭可能拦截自动化行为的安全/防护软件，或为本软件添加信任例外。

三、使用提示
--------------------------------
1. 请在确保遵守相关法律法规及目标软件/服务使用条款的前提下使用本软件。
2. 建议先在测试环境或非关键账号上验证脚本逻辑，确认无误后再用于正式环境。
3. 对于关键业务或高价值账号，请谨慎使用自动化脚本，并做好数据备份。

四、许可协议
--------------------------------
1. 在安装和使用本软件前，请仔细阅读安装程序中提供的《零界点 软件许可协议》（license.txt）。
2. 一旦继续安装，即视为您已阅读并同意该许可协议全部内容。

准备就绪后，请点击“下一步”继续安装。
